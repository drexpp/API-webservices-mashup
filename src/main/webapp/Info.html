Para la asignatura de arquitectura de integración de los sistemas de software(AISS), vamos a realizar un mashup haciendo uso de distintas APIs. El título de nuestro mashup es "MUSIC DIRECT", el objetivo de nuestro trabajo es aprender el uso de distintos servicios web así como ofrecer una aplicación web que pueda ser de utilidad para otras personas en el ámbito de la música. Haremos uso de Youtube para mostrar el video oficial de la canción buscada (Si es que tiene), usaremos Spotify como plataforma para mostrar información de artistas y álbumes. Mostraremos la letra de la canción que hemos buscado gracias a la API de Musixmatch y ademas sera traducida con la API de Google Translate. Por último tendremos la opción de añadir la canción buscada en nuestra playlist de spotify y en caso de querer añadir una nueva, tendremos la opción de crear una.

Tendremos, en principio, 4 vistas, una vista principal de búsqueda donde se nos dará la posibilidad de buscar por artista y canción, tendremos una segunda vista con los resultados de las llamadas a las APIs de Youtube, Itunes, Musixmatch y Google translate. Una tercera vista nos ofrecera la posibilidad de añadir una canción a nuestras playlists de spotify o crear una nueva playlist. Y por último tendremos una vista de error donde se nos informará en caso de que se haya producido un error, dandonos la posibilidad de volver a la pagina principal de búsqueda.
